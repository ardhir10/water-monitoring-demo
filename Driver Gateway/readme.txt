Cara Koneksi Websocket :
- Jalankan Program NodeJS 'run.bat' di Folder "Driver Gateway/Websocket"
- Setting Endpoint Websocket di Web "Setting/socket" 
--Tentukan Host/Endpoint socket Ex: "iotjpa.com" / Local address 
*Jika pakai local addres seperti localhost/IP Ex:192.168.2.8 Wajib menggunakan awalan http:// jadi http://192.168.2.8

Cara Jalankan Driver Modbus :
- Buka Driver gateway/Modbus Read lalu jalankan run.bat
